⚠️ Important: "Organizations" vs "Organization" Naming Issue

Your Catalyst Data Store currently has a table named "Organizations" (plural).
The application code references "Organization" (singular — no "s").
Catalyst does NOT support table renames. You must create a brand new table named exactly "Organization" and leave "Organizations" untouched (or delete it only after migrating any data).
